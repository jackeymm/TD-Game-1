1. 每次只执行 **一个 Task**
2. 每个 Task 必须可以独立运行验证

4. 必须限定修改文件范围
5. 每完成一个 Task 必须输出：

   * 修改文件列表

## 📦 Phase 1：项目初始化 + 游戏外壳

---

### Task 1.1 初始化项目结构

#### 目标

创建基础微信小程序结构

#### 要求

* 创建目录结构：

```
miniprogram/
  pages/game/
  utils/
  entities/
```

* 创建基础文件：

  * app.js
  * app.json
  * app.wxss
  * game 页面四件套

#### 验收

* 小程序可以正常打开
* 页面无报错

---

### Task 1.2 创建游戏页面基础UI

#### 目标

搭建游戏基础界面

#### 内容

* 顶部 HUD：

  * 生命
  * 金币
  * 波次

* 中间：

  * Canvas 区域（游戏区域）

* 底部：

  * 两个按钮：

    * 普通塔
    * 快速塔

#### 验收

* 页面结构正确显示
* 无报错

### Task 1.3 初始化游戏状态（GameState）

#### 内容

```js
}
```

#### 验收
